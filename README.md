# PlayGroundTemplate

言語やフレームワーク、ツールのお遊び用リポジトリのテンプレート

## Recommended directory structure

`docs/` メモ書きを入れるためのディレクトリ
`hogeProject/` 目的ごとにディレクトリを切る
`hogeProject/docs/` 目的ごとのメモ書きを入れるディレクトリ
`.editorconfig` スペースの量や文字コードを環境依存させないための設定ファイル
